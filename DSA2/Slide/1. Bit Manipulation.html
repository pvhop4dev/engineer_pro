1. Bitwise Operators

4/11

  • And:
    • 10101 & 11010 = 10000
  • Or:
    • 10101 | 11010 = 11111
  • Xor:
    • 10101 ^ 11010 = 01111
  • Not:
    • ~10101 = 01010
  • Shift right:
    • 10101 >> 2 = 101
    • n >> x = n div (2^x)
  • Shift left:
    • 10101 << 2 = 1010100
    • n << x = n * 2^x

  • Two’s complement:
    • First bit is the sign bit (1: negative)
    • -x = (~x) + 1
  • Example in 4 bits:
    • 0: 0000
    • 1: 0001
    • 7: 0111
    • -8: 1000
    • -7: 1001

2. Basic Bit Manipulation

7/11

7

def set(mask, i):

return mask | (1 << i)

​

def clear(mask, i):

return mask & (~(1 << i))

​

def toggle(mask, i):

return mask ^ (1 << i)

​

def get(mask, i):

return (mask >> i) & 1 == 1

def count(mask):

return mask.bit_count()

​

def isOdd(num):

return num & 1 == 1

​

def power2(n):

return 1 << n

​

def isPower2(n):

return n & (n - 1) == 0
