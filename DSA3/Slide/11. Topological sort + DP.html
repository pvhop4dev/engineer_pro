Topological sort

  • Topological sorting for DAG is a linear ordering of vertices such that for every directed edge u-v, vertex u comes before v in the ordering.
  • Find the topological sort of the example below.

​

1 4 2 3 5 6 7 => topo sort order

1 4 2 3 6 5 7 => topo sort order

​

1 2 5 4 3 6 7 => not a topo sort order

How to travel DAG by topological sort?

  • Define in_degree[v] is the number of non-visited nodes u such that u-v is an edge in the graph

​

  • Initiate in_degree array for all nodes of the graph
    • For edge u-v, increase in_degree[v] by 1.
  • Put all nodes v with in_degree[v] = 0 to a queue.
  • Take node from the queue one by one. For each node (call it as u):
    • For all adjacency v of u, decrease in_degree[v] by 1.
    • If in_degree[v] = 0, put v to the queue.
    • Continue the process until the queue is empty.
