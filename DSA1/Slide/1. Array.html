4/12

  • Static vs. dynamic arrays:
    • Static: Fixed size
      • C++, Java: int[], double[], etc.
    • Dynamic: Resizable (add/remove elements)
      • C++: vector
      • Java: ArrayList
      • Python: List
  • Data are stored contiguously in memory
  • Random access: O(1)
  • Adding/removing elements at the end (only dynamic arrays): O(1)
  • Adding/removing elements at arbitrary positions: O(n)

8/12

  • Dynamic Array, Adding/removing elements at the end (only dynamic arrays): O(1)
  • Java ArrayList / C++ vector
    • Internally has an fixed size array with default capacity as 10
    • Add means just operate on the end of the internal array, so O(1)
    • When internal array reach 50% capacity
      • Then create new array with higher capacity (usually 1.5 to 2 times the current capacity)
      • And copy elements from old array to new array
      • This operation is rare, so that’s why amortized time for add() still O(1)
